# Onboarding

Welcome to Sai! This guide will walk you through the essential steps to start trading.

## Prerequisites

Before you begin, make sure you have:

- A supported wallet (see [Wallet Setup](wallet-setup.md))
- Collateral tokens (see [Acquiring Collateral](acquiring-collateral.md))
- Gas tokens for transactions

## Quick Start

1. **Connect Your Wallet** → [Wallet Setup](wallet-setup.md)
   - Set up your EVM or WasmVM compatible wallet
   - Configure your network for Nibiru

2. **Get Collateral** → [Acquiring Collateral](acquiring-collateral.md)
   - Learn how to bridge or purchase collateral tokens
   - Understand valid collateral types

3. **Choose a Market** → [Choosing a Market](choosing-market.md)
   - Explore available trading pairs
   - Understand oracle pricing

4. **Understand Leverage** → [Leverage & Risk Management](../trading/leverage-and-liquidations.md)
   - Learn how collateral and leverage work
   - Understand liquidation mechanics

## Need Help?

- Having wallet issues? See [Wallet Setup](wallet-setup.md)
- Can't bridge assets? Check [Acquiring Collateral](acquiring-collateral.md)
- Questions about trading? Visit [FAQ](../learn/trading/faq.md)
